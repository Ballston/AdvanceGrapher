The following python program "AdvanceGrapher.py" is a gui wrapper
to help user view time series data, and fit ARMA models on the data.

Requirements:
numpy version 1.8.2
scipy version 0.13.3
pandas version 0.13.1
statsmodels version 0.6.1

The imported data needs to be in the csv format 
Date,Var1,Var2
03/31/2001,-1.1,1.4
06/30/2001,2.1,5.1
09/30/2001,-1.3,0.7
